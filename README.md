# $AlTeX$

Язык с упрощённым синтаксисом и структурой $\LaTeX$, транслируемый в код $\LaTeX$.

---

## Пример

Следующий код $AlTeX$

```
((alpha * 2) - 33) / 16 + beta - delta**2
```

транслируется в валиидный код $\LaTeX$

```latex
((\frac{(\alpha 2 - 33)}{16} + \beta) - \delta ^ 2)
```

и отобразится как:

$$((\frac{(\alpha 2 - 33)}{16} + \beta) - \delta ^ 2)$$
